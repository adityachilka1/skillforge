# Support

Need help with this project? Here's where to go:

## Bug reports & feature requests

Open a [GitHub issue](../../issues/new/choose) using the appropriate template. Include:

- The version you're using
- A minimal reproduction
- What you expected vs. what happened

## Questions & discussion

Use [GitHub Discussions](../../discussions) for open-ended questions, design discussion, or to share what you're building.

## Security reports

See [SECURITY.md](./SECURITY.md). Don't file a public issue for security reports.
